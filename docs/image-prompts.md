# astro-haze — Image generation prompts

各画像は下のプロンプトをそのまま ChatGPT / DALL·E に貼り付けて生成してください。
生成後、見出しに書いたパス・寸法・フォーマットで `public/images/...` に保存します（`public/` が無ければ作成）。フォーマットは projects=WebP / blog=JPG / og=PNG。

## メモ
- 生成後 `public/images/...` に置けば各ページに自動反映されます（参照パスはコンテンツ側に設定済み）。
- さらに最適化したい場合は画像を `src/assets/` に移して import すると Astro `<Image>` が AVIF/WebP に自動変換します。`public/` 直参照のままでも表示は問題ありません。
- favicon はベクターで用意済み（`public/favicon.svg`）。
